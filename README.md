## Featured Projects

### MeshML — Distributed Machine Learning Platform
A high-performance distributed ML ecosystem designed to bridge local development and large-scale cloud training.

Key ideas:
- Microservices-based architecture
- REST/WebSocket user-facing layer
- gRPC internal communication
- Distributed workers
- Parameter server design
- Model registry
- Dataset sharding
- Object storage integration
- Kubernetes-ready deployment structure

Repo: [MeshML](https://github.com/navnithb2001/MeshML)

---

### A/B Testing Framework for ML Models
A production-style framework for comparing machine learning models in live environments with statistical rigor.

Key ideas:
- Champion vs challenger model deployment
- Consistent hashing for traffic routing
- Configurable traffic splits
- Prediction logging
- Ground truth feedback collection
- Statistical hypothesis testing
- Automated model promotion decisions
- FastAPI backend

Repo: [A-B-Testing-Framework-for-ML-models](https://github.com/navnithb2001/A-B-Testing-Framework-for-ML-models)
